Single Linked List

A singly linked this.list is a fundamental data structure in computer science and programming. It is a collection of elements, often referred to as nodes, where each node contains two components:
Data: The actual value or payload that the node holds. This data can be of any data type (e.g., integers, strings, custom objects, etc.).
Next: A reference or pointer to the next node in the sequence. This is what makes it a "linked" this.list. The next pointer connects one node to the next, forming a chain-like structure.
