Double Linked List

A doubly linked list is a data structure that extends the concept of a singly linked list. Like a singly linked list, it is a collection of elements, or nodes, but each node in a doubly linked list contains three components:
Data: This is the actual value or payload held by the node. It can be of any data type.
Next: A reference or pointer to the next node in the sequence, just like in a singly linked list. This pointer connects the current node to the following node.
Previous: A reference or pointer to the previous node in the sequence. This is the key feature that distinguishes a doubly linked list from a singly linked list. The previous pointer allows you to traverse the list in both directions, forwards and backwards.
