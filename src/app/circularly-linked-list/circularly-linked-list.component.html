Circular Linked List

A circular linked list is a variation of a linked list data structure in which the last node of the list is connected back to the first node, forming a closed loop or circle. In a standard singly linked list, the last node typically points to null to indicate the end of the list, but in a circular linked list, it points back to the first node. Like a doubly linked list, it is a collection of elements, or nodes, that contains three components:
Data: This is the actual value or payload held by the node. It can be of any data type.
Next: A reference or pointer to the next node in the sequence, just like in a singly linked list. This pointer connects the current node to the following node.
Key characteristics of a circular linked list:
Infinite Loop: Because the last node points back to the first node, you can traverse the list indefinitely. There is no explicit end, as you can keep following the "next" pointers in a continuous loop.
No Null References: Unlike standard linked lists, where the last node points to null to indicate the end, circular linked lists do not have null references for the "next" pointers within the list.
